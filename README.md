# BLURRED — Personal Portfolio

A cinematic personal portfolio site with a loading screen, scroll-triggered username reveal, bio section, and social links.

## Tech Stack

- **TanStack Start** (React, file-based routing)
- **Tailwind CSS v4**
- **Google Fonts** — Syne (display) + Space Mono (mono)
- **Netlify** (hosting + edge functions)

## Features

- Animated loading screen with progress bar
- Full-viewport hero with username that starts blurred and unblurs as you scroll
- Scroll-triggered description and socials sections
- Dark void aesthetic with amber accent and noise texture overlay

## Running Locally

```bash
npm install
netlify dev
```

Open [http://localhost:8888](http://localhost:8888).

## Customization

Edit `src/routes/index.tsx`:

- Change `USERNAME` constant to your actual handle
- Update `SOCIALS` array with your real profile links and handles
- Edit the description text in the `desc-section`
